Connection between vMobile and Console Application

In Step 4 Configuration is required to establish connection between the vMobile and the Console Application. Once the vMobile connects to Console Application, the vMobile App is disconnected from the vMobile device.

Configure vMobile Name and Console IP
  • vMobile Name: The user should configure the vMobile device name (user-defined name) for which the Wi-Fi network is being configured

  • Console IP: Configure the Console IP address of the Console Application to which the vMobile is connected for further operations.

Note:Â To connect and configure another vMobile device, navigate back to Home screen and repeat the Steps 1 to 4 as listed above.

Control Options available on the (Step 4) Configuration screen
  • Back: click on the Back button in the app to navigate to the Step 3 Configuration screen

  • Done: click on the Done button in the app to connect vMobile to the Console Application (configured Console IP Address). The connection between the vMobile App and the vMobile device is disconnected.
