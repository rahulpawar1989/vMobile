vMobile™ Connection to Internet (Wi-Fi)

Step 3 Configuration is required to provide internet connection to vMobile™ device.

Connect my vMobile device to Wi-Fi
  • All the available Wi-Fi network connections in the proximity is displayed on the app screen.
  • User can select the Wi-Fi network from the list and provide the password in the pop-up window.
  • Tap on OK to save and send the Wi-Fi network connection details along with the password to vMobile.
  • Multiple Wi-Fi network connections can be configured and sent to vMobile.

Configure Wi-Fi network which is not listed (Manual Operation)
  • Tap on the ‘Do you want to configure a Wi-Fi network which is not listed?’ button on the screen
  • Enter the Wi-Fi network name to manually configure the Wi-Fi network, which is not listed
  • Also, Enter the Password to configure the network. Note: The password for the Wi-Fi connection is available in the vMobile Guide/Printed on the back of the vMobile device/is available in the vMobile SD.
  • Tap OK to save and send the new Wi-Fi network configuration details to vMobile. Tap Cancel to discard the network configurations and to navigate back to the available Wi-Fi list.
Control Options available on the (Step 3) Configuration screen
  • Back: click on the Back button in the app to navigate to the Step 2 Configuration screen
  • Next: click on the Next button in the app to navigate to the next configuration (Step 4) screen
